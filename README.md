# Marketplace KPI Dashboard

## Project Overview
This project analyzes marketplace performance metrics for a real-estate platform
using SQL and visualizes insights using Power BI.

## Datasets
- Users
- Listings
- Leads
- Transactions

## KPIs Implemented
- Total Users
- Daily New Users
- Monthly New Users
- Average Transaction Value
- Average Days to Sell
- Supply vs Demand Ratio

## Tools & Technologies
- PostgreSQL
- SQL
- Power BI
- Python
- GitHub

## Outcome
Designed and implemented business KPIs, executed analytical SQL queries,
and built an interactive Power BI dashboard to support data-driven decisions.
